## Model
The model implements the formulas described in the assignment notebook to create a transformer that mimics Shakespeare style dialogue. The parameters and design choices were defined in the assignment's configuration. 

## Evaluation
The training process ran for 2000 iterations. Overall the model was able to train to completion and mimic some Shakespearse style dialogue. However, there are occasions where the ideas and grammar of the model output is nonsensical. This could potentially be solved with longer training or more data

## Reflection
Overall, it was very interesting to see how a transformer can actually be implemented, since it is such a relevant topic to recent advances in technology. I had to run the training on a google colab GPU runtime to make sure the training completed within a reasonable time.
